Security — Pre-launch Checklist
OWASP Top 10 security checklist — verify all 10 before launch
25 Jun
The OWASP Top 10 is the industry-standard list of the 10 most common and critical web security vulnerabilities. AstroCRM's architecture is designed to cover all 10 — but they must be verified before going live. Claude runs through the checklist: (1) Access control — tenant isolation tests passing. (2) Cryptography — bcrypt + TLS + Supabase AES-256. (3) SQL injection — SQLAlchemy ORM throughout. (4) Security headers — CSP, HSTS, X-Frame-Options. (5) Rate limiting — slowapi on auth routes.
